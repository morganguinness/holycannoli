# Holy Cannoli (Netlify-ready)

## Local dev
npm install
npm run dev

## Build
npm run build
npm run preview

## Deploy to Netlify (Git import)
- Build command: `npm run build`
- Publish directory: `dist`

Included:
- `.nvmrc` (Node 20)
- `netlify.toml` (build + SPA redirect)
- Vite + React + TS + Tailwind + @vitejs/plugin-react
